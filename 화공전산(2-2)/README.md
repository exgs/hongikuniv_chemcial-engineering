### 읽어주세요

-   파이썬 코드 중에서 Excel 파일을 읽어오는 부분에서 에러가 날 수 있습니다.
   높은 확률로 MacOS에서는 폴더의 경로를 나타낼 때 `/`로 나타내고,  Window에서는 `\`로 나타내기 때문입니다.
   
      ```text
      Window: .\src\hello.py
      MacOS : ./src/hello.py
      ```
      읽어오는 파일의 경로를 확인해주세요!
